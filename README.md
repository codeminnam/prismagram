# Prismagram
Instagram clone with Express + Prisma + React and React Native

## User Stories

- [x] Create account
- [x] Request secret
- [x] Confirm secret
- [x] Like / unlike a photo
- [x] Comment on a photo
- [x] Search by username
- [x] Search by location
- [x] Follow user
- [x] Unfollow user
- [ ] See the full photo
- [ ] Edit my profile
- [ ] See user profile
- [ ] Upload photos
- [ ] Edit / delete post
- [ ] See the feed
