# Prismagram
Instagram clone with Express + Prisma + React and React Native

## User Stories

- [] Log in
- [] Like / unlike a photo
- [] Comment on a photo
- [] Search by username
- [] Search by location
- [] See user profile
- [] Follow / unfloow
- [] See the full photo
- [] Edit my profile
- [] Create account
- [] Upload photos
- [] Edit / delete post
- [] See the feed
